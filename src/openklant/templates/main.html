Registration component for the storage and disclosure of customer data according to the Customer Interactions and Contact Data API specifications.

Open Klant implements two APIs: the Klantinteracties API and the Contactgegevens API.
The Klantinteracties API is based on the VNG specification, which stores and accesses a limited set of data about customers and their interactions with the municipality. The Contactgegevens API is a unique API that is not based on a standard.
